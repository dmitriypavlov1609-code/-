# Инструкция: Как оформить выходной день

## Шаг 1: Определите дату

Убедитесь, что:
- Выходной нужен минимум через **48 часов** (2 дня)
- Вы не превысили лимит дополнительных выходных (4 в месяц)
- Дата не совпадает с пиковой загрузкой (праздники, выходные)

## Шаг 2: Отправьте заявку через бота

Напишите боту сообщение в свободной форме. Примеры:

**Простая заявка:**
```
Нужен выходной 15 апреля
```

**С указанием причины:**
```
Не смогу выйти на смену 20 числа, семейные обстоятельства
```

**На несколько дней:**
```
Хочу взять выходные с 10 по 12 мая
```

## Шаг 3: Дождитесь подтверждения

Диспетчер рассмотрит заявку и ответит в течение **24 часов**:

✅ **Одобрено** - выходной подтвержден, можете планировать дела

❌ **Отклонено** - будет указана причина отказа, можно:
  - Выбрать другую дату
  - Согласовать индивидуально с диспетчером
  - Поменяться сменой с коллегой

⏳ **На рассмотрении** - диспетчер еще не принял решение

## Шаг 4: Проверьте график

За день до выходного проверьте график:
- В общем чате водителей
- На доске в офисе
- Через бота (команда `/schedule` - будет добавлена позже)

## Частые ошибки

❌ **Поздняя заявка** - подаете за день до смены
  → Решение: подавайте минимум за 48 часов

❌ **Слишком много выходных** - превышен лимит
  → Решение: распределите выходные равномерно в течение месяца

❌ **Неясная формулировка** - "может быть нужен выходной"
  → Решение: указывайте конкретную дату и уверенность

❌ **Отмена выходного в последний момент** - "передумал"
  → Решение: планируйте заранее, частые отмены снижают приоритет

## Срочный выходной (экстренные случаи)

Если ситуация срочная (болезнь, форс-мажор):

1. Позвоните диспетчеру (не только бот!)
2. Уведомьте минимум за **4 часа** до начала смены
3. Укажите причину
4. Будьте готовы предоставить документы (больничный, справка)

Контакты диспетчера:
- Телефон: +7 (XXX) XXX-XX-XX
- Telegram: @dispatcher_username

## Советы

💡 Планируйте выходные заранее - больше шансов на одобрение

💡 Укажите причину (не обязательно, но помогает диспетчеру)

💡 Будьте гибкими - если одну дату не одобрили, предложите альтернативу

💡 Поддерживайте хороший рейтинг - водителям с высоким рейтингом выходные согласовывают чаще
